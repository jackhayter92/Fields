• Autofocus

    input type="text" autofocus

    Refresh the page and your cursor will be in this field.

  • Placeholder

    <input type="text" name="user-name" id="user-name" placeholder="First Name">

    The placeholder 'First Name' informs the user what needs to be inputted.

  • Required

    <input type="text" id="given-name" name="given-name" required>

    See what happens when you leave this field blank and click the submit button.

  • Pattern

    <input pattern="[0-9][A-Z]{3}" name="product" type="text" title="Single digit followed by three uppercase letters."/>

    You must conform to the pattern of a single digit followed by 3 letters in order to submit.

  • Multiple

    <select multiple>

    Select multiple options by holding control when clicking

  • Range

    <input type="number" min="1" max="10" class="field">

    The background will be red when the value is out of range and turn red when in range

  • Read Only

    <input type="text" value="UK" class="field" readonly>
